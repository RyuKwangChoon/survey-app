# 설문 조사 앱 (Survey App)

React로 개발된 설문 조사 웹 애플리케이션입니다.

## 주요 기능

- 설문 생성 및 관리
- 설문 참여
- 설문 결과 확인
- 다양한 질문 유형 지원 (텍스트, 단일 선택, 다중 선택)

## 기술 스택

- React
- TypeScript
- React Router
- Context API

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

## 프로젝트 구조

```
src/
  ├── components/     # 재사용 가능한 컴포넌트
  ├── context/        # Context API 관련 파일
  ├── types/          # TypeScript 타입 정의
  └── App.tsx         # 메인 애플리케이션 컴포넌트
```
